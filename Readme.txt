Задача:
Разработать приложение по поиску и группировке сообщений из твиттера по 
опредленному хештэгу. Твиты должны отображаться с автором, текстом и 
временем сообщения. Интерфейс должен состоять из поисковой строки для 
хештэга и областью отображения твитов (порядок, расположение и сортировка на Ваше усмотрение).
Можно использовать любые фреймворки. Приложение может быть как десктопным, так и вебовским.
Код необходимо выложить на github и выслать ссылку вместе с скомпилированным jar/war файлом.Инструкция по запуску приложения обязательна.

Решение задачи
Создано web jsf приложение.
Имеется поле для ввода хештега, и кнопка найти.

Запустить

Как запусть war file
Установить TomCat
Скопировать .файл война %apacha-tomcat-ver...%\webapps ( Напр.: C:\tomcat\webapps )
Запустить %apacha-tomcat-ver...%\bin\startup.bat
.War-файл будет извлечен автоматически в папку с таким же именем (без расширения)
Перейти к %apacha-tomcat-ver...%\conf\server.xml и взять порт для протокола http. <Connector port="8080" ... />. Значение по умолчанию - 8080.
Получить доступ к следующему URL:
localhost:port/twsearch/searchmain
